# UI Kit — DSLR Masterclass Web App

A high-fidelity recreation of the main screens of the DSLR Masterclass web app, built from the imported source in `source/`.

## Files
- `index.html` — interactive click-thru prototype (Home → Modules → Module Detail → Quiz)
- `Navbar.jsx`, `Hero.jsx`, `StatsRow.jsx`, `ModuleCard.jsx`, `ModuleGrid.jsx`, `SimCard.jsx`, `QuickLink.jsx`, `Badge.jsx`, `Button.jsx`, `SearchBar.jsx`, `FilterChips.jsx`

## Screens covered
1. **Home** — hero with SVG camera, stats row, learning-path grid, simulators, quick-links band
2. **Modules** — search + filter chips + 3-col card grid
3. **Module Detail** — header, difficulty badge, section progress
4. **Quiz** — question card with multiple choice

All components pulled from the source implementation; structure matches, with fake state for clickable flow.
